Magic the Gathering Web Application

This app allows users to create a personal database of their Magic the Gathering cards. Users can enter card information via a form, add them to the database, and edit or delete cards from their collection. The app also provides information about the game, including formats, rules, and where to play.

Technologies used in development:

  • Visual Studio IDE
  • C#
  • HTML, CSS, JavaScript
  • Bootstrap
  • Entity Framework
  • Identity Framework
