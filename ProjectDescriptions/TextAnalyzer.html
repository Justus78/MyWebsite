Text Analyzer GUI

This app allows users to analyze text to get the number of words, sentences, and unique words that are within the text. The user can type, paste in, or read text from a text file into the analyzer. The app provides the stats and gives the user a way to filter through the stats with a search bar. Users can choose between light, dark, and dark knight mode.

Technologies used in development:

  • Visual Studio IDE
  • C#
  • Windows Forms
