Greeneville Walks website

This website was a school project for Dr. Griffin at Walter State Community College. The main purpose of the site was to showcase student's work in Botany in the WSCC Herbarium and Symposium. The website also provides information about common trees and shrubs found around the walking trails in Greeneville Tennessee.

Technologies used in development:

  • VS Code IDE
  • HTML, CSS, JavaScript
  • PHPS
  • Bootstrap
